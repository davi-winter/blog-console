<strong>Titulo:</strong> Gerenciador de Blog

<strong>Descrição:</strong> Trata-se de um pequeno sistema em console para o gerenciamento de um blog, com funções CRUD através do micro-ORM Dapper.

<strong>Obs:</strong> Este pequeno projeto se trata de um exercício de desafio proposto pela plataforma de ensino <strong>balta.io</strong>, referente ao curso <strong>"Acesso à dados com .NET, C#, Dapper e SQL Server"</strong> 
com o mesmo parcialmente resolvido pelo instrutor.

<strong>Pré-requisitos e instalação:</strong>

1. Instalar o Visual Studio Code
2. Instalar o SQL Server
3.  Rodar o script em (../Scripts/blog.sql)

<strong>Tech Stack:</strong>

- Linguagem: C#
- Back-end: .NET com micro-ORM Dapper
- Banco de dados: SQL Server
- Front-end: Console
- Arquitetura: Repository Pattern
